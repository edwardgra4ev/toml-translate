# toml-translate

**toml-translate** — это инструмент для автоматического перевода TOML-файлов с английского языка на русский. Программа рекурсивно обходит указанную директорию, находит все файлы с расширением `.toml`, переводит значения выбранных ключей и сохраняет результат в отдельную папку, полностью сохраняя структуру каталогов.

Для перевода используется библиотека **Argos Translate**, работающая локально без обращения к внешним API. Благодаря использованию **tomlkit** сохраняется форматирование TOML-файлов, включая комментарии, порядок элементов и стиль оформления.

---

# Требования

* Python **3.11** или новее
* uv (рекомендуется для управления зависимостями)

Основные библиотеки проекта:

* `argostranslate`
* `tomlkit`

---

# Установка

Клонируйте репозиторий:

```bash
git clone https://github.com/edwardgra4ev/toml-translate.git
cd toml-translate
```

Установите зависимости проекта:

```bash
uv sync
```

Перед первым запуском необходимо установить языковую модель Argos Translate:

```bash
uv run python install_models.py
```

После установки модели программа готова к работе.

# Настройка

Все параметры находятся в начале файла `main.py`.

## Ключи для перевода

Список ключей, значения которых будут переведены:

```python
LIST_KEY = [
    "display_name",
    "description"
]
```

Программа переводит только строковые значения этих ключей независимо от глубины их вложенности.

При необходимости список можно изменить, добавив собственные ключи:

```python
LIST_KEY = [
    "title",
    "text",
    "tooltip",
    "display_name",
    "description"
]
```

---

## Исходная директория

Каталог, содержащий TOML-файлы для обработки:

```python
SOURCE_ROOT = Path("./StreamingAssets")
```

Программа рекурсивно просматривает все вложенные папки.

---

## Директория результата

Каталог, в который будут сохранены переведённые файлы:

```python
TARGET_ROOT = Path("./StreamingAssets_ru")
```

Структура каталогов полностью сохраняется относительно исходной директории.

Например:

```text
StreamingAssets/
├── Items/
│   └── Weapons.toml
└── NPC/
    └── Merchant.toml

↓

StreamingAssets_ru/
├── Items/
│   └── Weapons.toml
└── NPC/
    └── Merchant.toml
```

# Запуск

После настройки параметров выполните:

```bash
uv run python main.py
```

Во время работы программа выводит в консоль информацию о найденных файлах и ходе обработки.

---

# Как работает программа

1. Выполняется поиск всех файлов с расширением `.toml` в директории `SOURCE_ROOT`.
2. Каждый файл считывается с помощью библиотеки `tomlkit`.
3. Программа рекурсивно обходит всю структуру документа.
4. Если найден ключ из списка `LIST_KEY` и его значение является строкой, выполняется перевод с английского языка на русский.
5. Уже переведённые строки сохраняются во внутреннем кэше, поэтому одинаковые тексты переводятся только один раз.
6. Изменённый документ записывается в директорию `TARGET_ROOT` с сохранением исходной структуры каталогов.

---

# Результат

После завершения работы в каталоге `TARGET_ROOT` будут находиться переведённые TOML-файлы с сохранённой структурой директорий и форматированием документов.
